Installation Instructions:

  1. Download the APK file using the button above
  2. On your Android device, go to Settings > Security
  3. Enable "Unknown sources" (allow installation from unknown sources)
  4. Open the downloaded APK file to install the app
  5. Launch the app and enjoy!

App Features:

  • Manage construction sites and labor
  • Track attendance and payments
  • Generate weekly reports
  • Customer payment tracking
